Plugin Setup
Install Roblox Studio Plugin
1. Download StudioMindPlugin.lua
2. In Roblox Studio → Plugins → Manage Plugins
3. Click "Install from file" and select the .lua file
4. The plugin widget appears in your toolbar
5. Click Connect in the plugin to start syncing
6. Sync counter goes 1→10 on loop to confirm connection
⚠ Backend Note
For real plugin→site sync you need a backend relay server
(e.g. a simple Node.js/Express server or serverless function).
The plugin polls your server URL for pending tasks.
The site pushes tasks to the server after AI processes them.
